# Smart-Microwave-Oven-System-.github
Smart Microwave Oven System

Project Overview
A Python-based smart microwave oven system with GUI interface, recipe management, and advanced cooking features.

Team Members
Harrison Scgalski,
Arturo Menchaca,
Oladipo Towobola and
Oliver Michael Wilderman



 Features
1. Graphical User Interface - User-friendly control panel with multiple tabs
2. Smart Cooking Controls- Multiple power levels, cooking modes, and timer
3. Recipe Management- Store, search, and favorite recipes
4. Auto Defrost - Calculate defrost times based on food type and weight
5. Cooking History - Track previous cooking sessions
6. Real-time Status - Monitor microwave status and sensor readings



Requirements
- Python 3.7+
- tkinter (usually comes with Python)
- numpy (optional, for sensor simulation)
